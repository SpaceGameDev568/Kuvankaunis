# Kuvankaunis

A modern Minecraft modpack focused on visuals and exploration.

https://modrinth.com/modpack/kuvankaunis

## Build / Export 

This pack is compiled using [Packwiz](https://packwiz.infra.link/), and can be exported to a modrinth pack by running `packwiz mr export`.
